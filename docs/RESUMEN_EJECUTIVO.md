# Care360

## Resumen Ejecutivo

Care360 es una aplicacion movil Android orientada a la coordinacion del cuidado familiar de una persona dependiente o mayor. Su propuesta de valor es centralizar en un unico espacio la informacion esencial que normalmente se dispersa entre mensajes, llamadas, notas y agendas personales.

El producto permite organizar la colaboracion entre familiares o cuidadores alrededor de un grupo compartido. Dentro de ese grupo se gestionan flujos clave como medicacion, citas, incidencias, actividad reciente, datos de perfil y contexto de salud.

## Problema Que Resuelve

En el cuidado familiar aparecen de forma recurrente estos problemas:

- falta de coordinacion entre miembros de la familia
- duplicidad de informacion o informacion contradictoria
- olvidos de medicacion o citas
- dificultad para seguir incidencias y cambios importantes
- ausencia de una vista comun sobre el estado general de la persona cuidada

Care360 busca resolver estos puntos con un enfoque simple: un grupo, una persona cuidada y una base comun de informacion util y accionable.

## Propuesta de Valor

- coordina a varias personas sobre un mismo caso de cuidado
- centraliza tareas y datos clave en un flujo sencillo
- reduce la dependencia de WhatsApp, notas y memoria informal
- facilita seguimiento, contexto y continuidad en el cuidado
- ofrece una base clara para evolucionar hacia un producto de salud/care tech mas completo

## Estado Del Proyecto

Care360 se encuentra en estado de MVP funcional avanzado sobre Android nativo. El nucleo principal del producto ya esta implementado y permite demostrar claramente el valor de la solucion.

Modulos ya presentes:

- gestion de grupos familiares
- alta y union a grupos por codigo
- perfil de la persona cuidada
- medicacion
- citas
- incidencias
- actividad reciente
- patologias
- seguimiento
- familia
- modo SOS
- soporte multilenguaje base ES/EU

## Posicionamiento Actual

El proyecto esta bien orientado para presentarse como:

- idea de producto con base funcional real
- MVP avanzado
- activo tecnico para acelerar un producto comercial
- base para continuidad por parte de una startup, agencia o equipo tecnico

## Decision De Producto Sobre Foto De Perfil

La gestion de foto de perfil de la persona cuidada se ha dejado fuera del flujo visible actual de forma deliberada para priorizar estabilidad del nucleo funcional. Se considera una mejora futura, no un elemento central del valor del producto en su estado actual.

## Oportunidad

Care360 tiene recorrido para evolucionar en varias direcciones:

- app familiar de coordinacion de cuidados
- producto premium para familias cuidadoras
- solucion white label para asociaciones o servicios de acompanamiento
- producto ampliable hacia seguimiento de salud domiciliaria

## Mensaje Corto De Presentacion

Care360 es una app para coordinar el cuidado familiar de una persona mayor o dependiente desde un unico espacio compartido: medicacion, citas, incidencias, perfil y actividad reciente, con una base funcional ya construida sobre Android.
