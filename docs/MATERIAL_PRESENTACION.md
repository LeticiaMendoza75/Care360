# Care360

## Material De Presentacion

## 1. Pitch Corto

Care360 es una app Android para coordinar el cuidado familiar de una persona mayor o dependiente desde un unico espacio compartido. Centraliza medicacion, citas, incidencias, perfil y actividad reciente para reducir desorden, olvidos y falta de coordinacion entre familiares.

## 2. Mensaje De Apertura

Hoy muchas familias gestionan el cuidado con mensajes, llamadas, notas y recuerdos informales. Care360 propone unificar todo eso en una app con estructura clara, foco operativo y una base funcional ya construida.

## 3. Problema En Una Frase

Las familias cuidadoras no suelen tener una herramienta unica, simple y compartida para coordinar el dia a dia del cuidado.

## 4. Solucion En Una Frase

Care360 convierte el cuidado familiar en un flujo compartido y organizado: grupo, perfil, medicacion, citas, incidencias, actividad y seguimiento.

## 5. Puntos Fuertes A Destacar

- caso de uso facil de entender
- necesidad real
- colaboracion por grupos
- arquitectura ordenada
- MVP funcional ya existente
- recorrido claro de evolucion

## 6. Lo Que Conviene Decir

- es un MVP funcional avanzado
- el nucleo del valor ya esta implementado
- el producto esta orientado a coordinacion familiar del cuidado
- la base tecnica permite evolucion futura
- ciertas mejoras no nucleares se han dejado fuera del alcance visible para priorizar estabilidad

## 7. Lo Que Conviene No Decir

- que esta totalmente terminado
- que ya es multiplataforma
- que reemplaza sistemas clinicos
- que todas las mejoras futuras estan cerradas
